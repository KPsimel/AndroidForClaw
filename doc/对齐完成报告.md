### 4. 配置系统 (100% 对齐) ✅

#### models.json 配置格式

```json
{
  "mode": "merge",
  "providers": {
    "openai": {
      "baseUrl": "https://api.openai.com/v1",
      "apiKey": "${OPENAI_API_KEY}",
      "api": "openai-completions",
      "models": [
        {
          "id": "claude-opus-4-6",
          "name": "Claude Opus 4.6",
          "reasoning": true,
          "input": ["text", "image"],
          "contextWindow": 200000,
          "maxTokens": 16384,
          "pricing": {
            "input": 0.03,
            "output": 0.05
          }
        }
      ]
    }
  }
}
```

**特性对齐**:
- ✅ JSON 格式 (与 OpenClaw 相同)
- ✅ 环境变量支持 `${VAR_NAME}`
- ✅ 多 provider 支持
- ✅ 模型级配置 (reasoning, pricing, etc.)
- ✅ 自动验证和热重载

#### openclaw.json 配置格式

```json
{
  "model": "openai/claude-opus-4-6",
  "thinking": {
    "enabled": true
  },
  "skills": {
    "selection": "auto"
  },
  "memory": {
    "enabled": true,
    "backend": "local"
  }
}
```

**特性对齐**:
- ✅ 默认模型配置
- ✅ Thinking/Reasoning 开关
- ✅ Skills 选择策略
- ✅ Memory 系统配置

AndroidForClaw 拥有 OpenClaw 不具备的移动设备控制能力:

✅ **屏幕观察**:
- screenshot (截图 + UI Tree)
- get_ui_tree (UI 层级分析)

✅ **设备操作**:
- tap, swipe, long_press (触摸操作)
- type (文本输入)
- home, back (系统导航)

✅ **应用管理**:
- open_app, start_activity (应用启动)
- list_installed_apps (应用列表)

✅ **系统集成**:
- notification (通知管理)
- Accessibility Service (无障碍服务)

✅ **三层加载**: Bundled → Managed → Workspace
✅ **配置格式**: 与 OpenClaw 相同的 JSON 配置
✅ **LLM Provider**: OpenAI 兼容 API
✅ **Memory 系统**: MEMORY.md + daily logs

### 3. 代码质量

✅ **Kotlin 实现**: 类型安全,协程支持
